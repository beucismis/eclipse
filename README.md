# eclipse
A dark theme for [Zola](https://getzola.org).

## Installation

First download this theme to your `themes` directory:

```
cd themes
git clone https://github.com/beucismis/eclipse
```

and then enable it in your `config.toml`:

```
theme = "eclipse"
```
